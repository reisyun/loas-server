# Loas

애니메이션을 탐색하고, 나만의 리스트에 기록해서 애니를 추천받을 수 있는 웹앱

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```

## Stack

- Typescript
- GraphQL
- NestJS
- Express
- Prisma
